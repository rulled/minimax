Privacy Policy

MiniMax TTS Automation · Last updated: 2026-04-08

MiniMax TTS Automation is a Chrome extension that automates workflow steps on the MiniMax text-to-speech page.

What the extension does

The extension helps a user load or paste text scripts, assign voices and languages, automate generation on the MiniMax TTS page, and download generated MP3 files with organized names.

Data the extension processes

The extension may process text that the user uploads or pastes, speaker labels, voice mappings, file names, download history, and runtime state required to manage an automation session.

Where data is stored

The extension stores its own working data locally in Chrome extension storage on the user's device. This may include voice mappings, selected language, skipped entries, batch queue state, download history, and UI state.

Network activity

The extension works only on https://www.minimax.io/audio/text-to-speech*. It does not send user data to a separate server controlled by the extension developer. It automates actions inside the MiniMax page already opened by the user and uses Chrome's local downloads API to save generated files.

MiniMax, as the website/service used by the user, may process entered text according to its own terms and privacy policy.

Downloads

When the user runs the automation, the extension saves generated MP3 files to the user's device using Chrome's downloads functionality.

Sharing of data

The extension does not sell user data and does not share user data with third parties for advertising or analytics purposes.

Security

The extension uses a limited permission set and restricted host access only for the MiniMax TTS page required for its core functionality.

User control

  • Remove the extension from Chrome.
  • Clear extension history/state from the extension UI where available.
  • Remove downloaded files from the local device.
  • Clear browser extension storage through Chrome.
