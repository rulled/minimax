# Reviewer Notes

## Extension purpose
This extension automates repetitive text-to-speech work on the MiniMax TTS page and saves generated MP3 files locally.

## Where it works
Only on:

`https://www.minimax.io/audio/text-to-speech`

## Why login is required
MiniMax TTS is an authenticated web app. The extension cannot demonstrate its core workflow unless the user is signed in to MiniMax.

## Suggested review flow
1. Sign in to MiniMax.
2. Open `https://www.minimax.io/audio/text-to-speech`.
3. Click the extension icon.
4. Upload a Markdown dialogue file or paste a short script.
5. Assign voices if needed.
6. Start automation and confirm that generated MP3 files are downloaded locally.

## Important note for submission
If Chrome Web Store review requires credentials, provide a dedicated reviewer test account in the review notes field of the dashboard. This repository does not include account credentials.
