# Privacy Policy

Last updated: 2026-04-08

`MiniMax TTS Automation` is a Chrome extension for automating voice generation workflows on the MiniMax text-to-speech page.

## What the extension does

The extension helps a user:

- load or paste text scripts,
- assign voices and languages,
- automate generation on the MiniMax TTS page,
- download generated MP3 files with organized names.

## Data the extension processes

The extension may process:

- text that the user uploads or pastes into the extension,
- speaker labels and voice mappings created by the user,
- file names and download history related to generated audio,
- runtime state needed to resume or manage an automation session.

## Where data is stored

The extension stores its own working data locally in Chrome extension storage on the user's device. This can include:

- voice mappings,
- selected language,
- skipped entries,
- batch queue state,
- download history,
- UI state for convenience.

The extension does not operate its own remote database for this data.

## Network activity

The extension works only on `https://www.minimax.io/audio/text-to-speech*`.

The extension does not send user data to a separate server controlled by the extension developer. It automates actions inside the MiniMax page that the user already opened and uses Chrome's local downloads API to save generated files.

Please note that MiniMax, as the website/service used by the user, may process the entered text according to its own terms and privacy policy.

## Downloads

When the user runs the automation, the extension saves generated MP3 files to the user's device using Chrome's downloads functionality.

## Sharing of data

The extension does not sell user data.

The extension does not share user data with third parties for advertising or analytics purposes.

## Security

The extension uses a limited permission set and restricted host access only for the MiniMax TTS page required for its core functionality.

## User control

The user can:

- remove the extension,
- clear extension history/state from the extension UI where available,
- remove downloaded files from the local device,
- clear browser extension storage through Chrome.
